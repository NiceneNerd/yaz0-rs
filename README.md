[![Crates.io Version](https://img.shields.io/crates/v/yaz0.svg)](https://crates.io/crates/yaz0)

## tools
To install `yaztool`, a de/flating utility for yaz0 files, do
```
$ cargo install yaz0 --features=yaztool
```
